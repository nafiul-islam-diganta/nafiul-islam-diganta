### 🚀 Featured Project

**[Zabbix 7.0 LTS — NOC Monitoring Lab](https://github.com/nafiul-islam-diganta/zabbix-monitoring-lab)**

Built a production-style infrastructure monitoring server from scratch — no tutorials, just Linux logs and systematic troubleshooting.

- 392 monitoring items · 226 triggers · 3 hosts across Linux and Windows
- Trigger logic using time-window averaging (`avg()`) to cut false-positive alerts
- End-to-end email alerting, validated against a live simulated service outage
- Role-based access control (Super Admin / read-only NOC Analyst)
- Diagnosed and resolved a carrier-grade NAT/client-isolation issue via hypervisor-level port forwarding

**Stack:** Zabbix 7.0 · Ubuntu Server 24.04 · MySQL 8 · Nginx · VMware Workstation Pro
